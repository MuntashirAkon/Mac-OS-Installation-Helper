El Capitan Installation Helper

Applied only for laptops with Clover EFI and GPT. See this post for details.

last updated on 31st December, 2015 by Muntashir.

Introduction

Since Apple does not want hardwares, other than their own, support their Operating System. It is a bit tricky to install Mac OS to a unsupported system. Hackintoshers are trying hard for years to get it used in unsupported systems. But still there are many hardwares that is not supported and some hardware may never support Mac OS. El Capitan Installation Helper is a tool that can help you install Mac OS to any supported laptops.

If you are using this tool, be sure to issue the result. Your simple help may help a thousand others.

Requirements

  • Supported Laptops (if you don't know Google it)
  • A Mac/Hackintosh (you can easily borrow it)
  • Working internet connection
  • A 16GB USB pendrive (may work on 8GB, too)
Note:
  • Know your hardwares. You can check your user manual or other apps, like CPU-Z.
  • This may be a clean install, as sometimes windows is messed up with Mac. So, Backup your data.
  • This method only works for the laptops which have support for EFI with GPT.
  • There is always a possibilty of messing something up. So, you should have another pendrive with bootable ubuntu or just a live cd.
  • Be calm and never miss a step. This method is easier than any other methods available.

Installation


  1. Download the usb_helper.sh. Let's imagine you downloaded the file in the ~/Downloads directory.
  2. Insert the USB pendrive.
  3. Open Terminal.app located at /Applications/Utilities & run these commands: cd ~/Downloads
    chmod 755 usb_helper.sh
    ./usb_helper.sh
  4. If everything works fine, you will have a bootable USB.
  1. Insert the pendrive to one of the USB ports.
  2. Boot your computer to that USB device. The mechanism varies here, with most computers using one of the F1..F12 keys, Esc key, or (newer computers) have a dedicated power button for boot device selection, BIOS settings, etc. Refer to your user manual if necessary.
  3. Clover bootloader screen shows up, select install_osx.
    [Note: You can press spacebar at this point to change how Clover boots OS X (verbose, without caches, etc). Verbose is a good idea when you're not certain if it is going to work.]
  4. Press spacebar (or plug in a USB keyboard) if a nagging screen about Mouse/keyboard appears. Install screen will appear, use Disk Utility to format your drive as GPT, create a Mac OS X Extended (Journaled) partition, name it Macintosh HD and install OS X to the Macintosh HD partition. The installer will extract necessary files and verify them against the USB. This will not install anything into the target partition. This takes a lot of time at the end, though you only see "a minute remaining".
    [Note: You may not able to format any partition and will get an error. In that case, there's no other option but to make a clean install (i.e. Format the whole drive and install OS X)].
  5. Restart and again boot to your USB as you did in step #2.
    [Note: There is slight chance that the USB doesn't show up in the BIOS Menu. In that situation, you might need a bootable Ubuntu (or, other linux distro) to copy the bootloader in the EFI partition of your hard drive. The tutorial will come very soon.]
  6. Clover bootloader screen shows up, select install_osx (NOT Boot OS X from Macintosh HD) as you did in step #3.
  7. Install screen will reappear and continue the installation process. This time, installer will install files to your target partition and create Recovery HD partition.
  8. Restart and again boot to your USB as you did in step #2 and step #5.
  9. Clover bootloader screen shows up, select Boot OS X from Macintosh HD.
  10. If everything goes well, you will finish up the installation process and arrive at the OS X desktop.
  11. Just like pendrive setup, download hdd_helper.sh from below and follow the same instructions. But this time the file name will be changed only.

If you don't find your graphics card, you have to manually edit the config.plist file yourself.

One kext may work for multiple ethernet. Google your ethernet for more information.
